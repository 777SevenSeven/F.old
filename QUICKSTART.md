# Quickstart

## Backend

```bash
pip install -r requirements.txt
playwright install chromium
python run.py --mode api
```

## Frontend

```bash
cd frontend
flutter pub get
flutter run -d chrome
```

## Demo Flow

1. Create a preference in the UI (Chat ID, product, city, price range).
2. Click search to see mock results instantly.
3. Use AI buttons to generate quick analysis and search links.
